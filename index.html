About the vault

This is a file storage vault. It is meant to store games ISOs and other files that are being taken down from the internet en masse.

This is a private server. If you see this page, you've already been authenticated and you have been given access. There is also a login for this specific website but I don't use it, so you may just ignore it. No features are kept from you buy not having an account.

Bigger games MUST be zipped. The server will check for a .zip file format. If you try to upload a game and it doesn't work, try zipping it first.

Follow the instructions bellow to keep this thing organized and we'll all have a good time :-)

How to use the Vault

How to upload content
Step 0 (games only)

Go to the correct folder for the console of the game you want to upload

Step 1

Create a subfolder for the game/movie you want to upload. Click on "upload" then "create folder" and enter the name of the game/movie. For movies, also add the year after the title of the movie in parentheses. Example : "Dune: Part One (2021)"

Step 2

Navigate inside the newly created folder and click on "upload". This time, select "pick a file" and select the file you wish to upload. For larger games, please make sure it’s a .zip archive. No need to do the same for movies.

Step 3

Once the file is selected, click on “send x files” to upload the file. This may take some time for larger files.

How to download content
Step 1

Navigate to the parent folder of the content you wish to download.

Step 2

Click on the file you want to download.

Step 3

In the pop-up box, click Download.
